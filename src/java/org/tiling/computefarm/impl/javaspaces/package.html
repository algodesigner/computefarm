A JavaSpaces implementation of ComputeFarm.

The following system properties are available to the ComputeSpace client (JobRunner):

  • org.tiling.computefarm.impl.javaspaces.DontReloadClasses - setting this to true to true will cause the JavaSpaces JobRunner to not send a reload classes signal to workers (defaults to false)
  • org.tiling.computefarm.impl.javaspaces.MaxResultLifetime - the maximum time, in milliseconds, that a result object can remain in the JavaSpace before being reaped (defaults to 5 minutes)

The following system properties are available to the ComputeSpace worker:

  • org.tiling.computefarm.impl.javaspaces.MaxTaskExecutionTime - the time, in milliseconds, after which the task is aborted and returned to the space (defaults to 5 minutes)
